------------------------------------------------------------------------------
KNOWN LIMITATIONS TO DOCUMENT
------------------------------------------------------------------------------
- ECOD is row-level; LOO bridge assumes each cell contributes independently.
  This under-counts multi-cell intentional edits (e.g., TFM masks 3 cells).
- TabFact has no cell-level ground truth — row-level score only.
- TwitterBot-LLM is single-class (all INT) — only precision/recall for INT
  can be reported; F1-unint is undefined.
- Threshold tuned on same distribution as test (mild optimism); use with care.
- ECOD trained on clean data only — does not see any dirty rows at train time.
